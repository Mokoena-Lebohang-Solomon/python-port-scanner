# Python Port Scanner

## Description

This project is a simple network port scanner written in Python.  
It scans a target machine to identify open ports.

Port scanning is a common technique used in cybersecurity and penetration testing to discover services running on a network.

## Features

- Scans ports from 1 to 1024
- Identifies open ports
- Demonstrates basic network reconnaissance

## Technologies Used

- Python
- Socket library

## How to Run

1. Clone the repository

2. Run the program

python port_scanner.py

3. Enter a target IP address when prompted

Example:
127.0.0.1

## Example Output

Scanning target: 127.0.0.1

Port open: 80  
Port open: 443

## Cybersecurity Concept

Port scanning helps identify services running on a machine. Security professionals use this information to find vulnerabilities in networks.
